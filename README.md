<h2>Installation</h2>
To run Store Rating App locally, follow these steps:

<h3>1. Clone the repository:</h3>

<div class="bg-light p-3 rounded border">
  <pre class="mb-0">
    <code>git clone https://github.com/yourusername/StoreRatingApp.git</code></pre>
</div>

<h3>2. Navigate to the project directory:</h3>

<div class="bg-light p-3 rounded border">
  <pre class="mb-0">
    <code>cd StoreRatingApp</code></pre>
</div>
<h3>3. Install the dependencies for the frontend:</h3>

<div class="bg-light p-3 rounded border">
  <pre class="mb-0">
    <code>npm install</code></pre>
</div>
<h3>4. Set up environment variables:</h3>

Create a .env file in the root directory and configure the following variables:

<div class="bg-light p-3 rounded border">
  <pre class="mb-0">
    <code>DB_PASSWORD=&lt;your_db_password&gt;</code></pre>
</div>
<div class="bg-light p-3 rounded border">
  <pre class="mb-0">
    <code>DB_NAME=&lt;your_db_name&gt;</code></pre>
</div>

<h3>5. Run the development server:</h3>

<div class="bg-light p-3 rounded border">
  <pre class="mb-0">
    <code>npm run dev</code></pre>
</div>
<h3>6. Open your browser and navigate to:</h3>

<div class="bg-light p-3 rounded border">
  <pre class="mb-0">
    <code>http://localhost:5173</code></pre>
</div>
<h2>Usage</h2>
Once the application is running, you can:

<h3><li>System Admin Dashboard:</h3> Manage users, stores, and platform statistics with full sorting and filtering capabilities.</li>
<h3><li>Normal User Dashboard:</h3> Browse, search, and rate stores while managing personal profile settings.</li>
<h3><li>Store Owner Dashboard:</h3> View all user ratings and monitor the average performance of your store.</li>
